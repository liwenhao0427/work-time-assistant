# 工时智能校验助手 (Work Time Smart Verification Assistant)

这是一个基于 Vue 3 + Vite + Tailwind CSS 开发的工时智能校验助手应用。

## 功能特性

- **智能校验**: 自动校验工时数据的准确性
- **现代化 UI**: 使用 Tailwind CSS 构建的响应式界面
- **高效开发**: 基于 Vite 的极速开发体验

## 技术栈

- [Vue 3](https://vuejs.org/) - 渐进式 JavaScript 框架
- [Vite](https://vitejs.dev/) - 下一代前端构建工具
- [Tailwind CSS](https://tailwindcss.com/) - 原子化 CSS 框架
- [TypeScript](https://www.typescriptlang.org/) - 强类型 JavaScript 超集

## 本地开发

1. **安装依赖**

   ```bash
   npm install
   ```

2. **启动开发服务器**

   ```bash
   npm run dev
   ```

3. **构建生产版本**

   ```bash
   npm run build
   ```
